# Book Store Management System

## Overview
A SQL-based Book Store Management System designed to manage books, customers, orders, and inventory.

## Features
- Manage books and authors
- Manage customers
- Track orders
- Inventory management
- Sales reports and analysis

## Database Concepts Used
- Primary Keys
- Foreign Keys
- Joins
- Aggregate Functions
- GROUP BY
- HAVING
- ORDER BY

## Technologies Used
- SQL
- MySQL
